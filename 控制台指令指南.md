# Slay the Spire 2 控制台指令指南（LocalMultiControl 联调用）

> 适用范围：用于本地联调时快速跳转流程、复现问题、验证修复。  
> 说明：命令来源于 `src/Core/DevConsole/ConsoleCommands/` 的实现。

## 你现在最关心的两个命令

1. 触发“藏宝图”事件（TheLegendsWereTrue）

```text
event THE_LEGENDS_WERE_TRUE
```

2. 跳到第二幕（第二地图）

```text
act 2
```

## 常用命令速查

1. 跳到指定事件

```text
event <事件ID>
```

示例：

```text
event THE_LEGENDS_WERE_TRUE
event WELCOME_TO_WONGOS
```

2. 跳到指定幕（按序号）

```text
act <幕序号>
```

示例：

```text
act 1
act 2
act 3
```

3. 把当前房间改为指定类型

```text
room <房间类型>
```

示例：

```text
room Event
room Treasure
room RestSite
room Combat
```

4. 地图自由旅行开关（可在地图上任意跳点）

```text
travel
```

## 与本 Mod 联调的推荐顺序

1. `act 2`（先进入第二幕环境）
2. `event THE_LEGENDS_WERE_TRUE`（拿到藏宝图任务）
3. 继续推进到宝箱节点，验证“每个角色都移除藏宝图并各得 600 金币”

## 备注

1. `event` 命令参数是事件 ID（大写下划线），不是中文事件名。
2. `act 2` 是“直接切到第二幕”的最快方式，适合回归测试。
